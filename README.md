# Starboard

A github stared repo management app

## Tech Stack

- Database
    - Postgres 9.5 (primary datastore)
    - Redis 3.0 (session store)
- Server
    - Node.js
    - Koa (http server)
    - React (server rendering)
- Browser
    - React
    - React Router
    - Redux
- Support
    - Docker
    - Webpack
    - Babel
    - Eslint
